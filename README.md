# 개발 환경 셋팅

**babel** - .ts 파일을 ES5 버전으로 트랜스파일

**typescript** - js의 런타임 버그 방지를 위한 타입 지정

**tsc** - 트랜스파일전 타입 체크용으로만 설정

**webpack** - 번들링으로 인한 코드 최적화

**jest** - 테스트코드 작성을 위함
